# AtzenBot
Der "Die Atzen" Discord Bot. Dieser wird sämtliche wichtigen funktionen, welche für den Server von nöten sind, inne haben.
##                                             Allgemeine Inhalte:
- Mute System
- Warn System
- Ban System
- Serverinfo
- Umfragen
- Logging
- Benutzerdefiniertes
- Roulette
- Blackjack

                                                   Und vieles Mehr!
